# 🌿 Feel Better Web App

A full-stack web application built to help users improve their mental well-being by tracking moods, writing journal entries, and receiving motivational content.

---

## ✨ Features

- 📝 **Daily Journaling** – Write and store your thoughts
- 😊 **Mood Tracking** – Track your emotional state over time
- 💡 **Motivational Content** – Get positive and encouraging messages
- 📊 **User-friendly Dashboard** – Simple and clean interface
- 🔐 **Secure Backend API** – Built with Laravel

---

## 🛠️ Tech Stack

### Backend

- Laravel (PHP)
- MySQL Database
- REST API

### Frontend

- React
- Vite
- JavaScript (ES6+)

## 🚀 Installation Guide

### 1. Clone the repository

```bash
git clone https://github.com/Queen4087/Feel-Better-Web-App.git
cd Feel-Better-Web-App
```

Backend Setup (Laravel)
cd backend

composer install
cp .env.example .env

php artisan key:generate
php artisan migrate

php artisan serve
Frontend Setup (React)
cd frontend

npm install
npm run dev
